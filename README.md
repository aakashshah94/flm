# flm

An online portal used by the college faculties to apply for leave to head of the department and the head of the department can also aproove or disaproove the application based on the credits remaining of the faculty.

Users:
-Login activity for each type of user
1>Faculty:
    Features:
            -Dashboard
            -Apply for leave
            -Check balance leaves(laves remaining)
            -Get notified when leave application gets aprooved or disaprooved
            -Check application status
         
2>Head of the department (admin):
            -Dashboard
            -Get notified when new application arrives
            -Check aprooved, disaprooved and pending applications
            -Add faculty to the system
            -Aproove or disaproove the leaveapplication
            -check balance leaves of each faculty
